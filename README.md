# Codex Agent Template

A compact starter set for repo guidance.

Files:
- `AGENTS.md`: root rules for all coding work
- `.codex/AGENTS.md`: shorter Codex-specific reminder
- `.codex/PLANS.md`: quick planning scaffold
- `.codex/commands/feature.md`: reusable feature workflow
- `.codex/checklists/ship.md`: pre-push checklist
- `.codex/config.toml.example`: optional Codex config example
